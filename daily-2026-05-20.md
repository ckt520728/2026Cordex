## 技術筆記（踩坑與 learnings）

1. **Anaconda PIL `DLL load failed` 根因不是 Pillow 壞** — 是 conda `Library\bin`（libjpeg/zlib/libssl…）沒掛 PATH，只有 `conda activate` 才會掛。治本：`import enable_conda_dlls`。**不要重灌 Anaconda。** 同根因也造成 `pip` 的 `ssl module not available`。
2. **光靠 `os.add_dll_directory(...Library\bin)` 不夠** — Pillow `_imaging` 有傳遞相依，必須**同時注入 `os.environ["PATH"]`**，實測才成功。
3. **PowerShell 5.1 把 UTF-8（無 BOM）腳本當 ANSI 讀** → 中文亂碼且報詭異 parser error。解：`.ps1` 全 ASCII，中文走 UTF-8 sidecar，用 `[System.IO.File]::ReadAllText(..., UTF8)` 讀。
4. **含中文路徑** PowerShell `Test-Path`/`Join-Path` 失敗（同上根因）→ 先用 Bash `cp` 到 ASCII 路徑，再用 `-LiteralPath`。
5. **python-pptx 退路** = import 前注入 PIL stub（涵蓋 Image/ImageFont/ImageDraw，用 struct 解 PNG/JPEG header）。
6. **PowerPoint COM 不穩**：`msoTriState` 要用 `-1/0` 不能 `$true/$false`；`RPC_E_CALL_REJECTED` 要 sleep+retry。結論：python-pptx + stub 比 COM 穩。
7. **PPTX 檔案鎖（`~$` owner file）** → 加 fallback 檔名 `try save except PermissionError: save_new`。
8. **HTML 簡報排版**：圖被裁是因為固定 height + `overflow:hidden`；改用 `aspect-ratio` + `max-height` + `object-fit:contain`，並 override 該頁 inner 的 max-height。
9. **HTML 圖一律 base64 內嵌**，不要相對路徑。
10. **git push from PowerShell** 進度走 stderr，`$ErrorActionPreference=Stop` 會誤觸 → 暫設 Continue + 看 `$LASTEXITCODE`。
11. **claude-skills repo 分支是 `master` 不是 `main`**；push 前若遠端有 web 上傳的 commit，先 `git rebase origin/master`。
12. **.skill 打包**：用 `System.IO.Compression` + 正斜線 entry，單一根目錄含 SKILL.md（不能用 `Compress-Archive`）。
