# PDF 報告解剖

## 工作流程

1. 逐頁檢視正文、圖、表、caption、註腳與附錄；記錄頁碼。
2. 先確認文件期間、母體、單位、幣別與比較基準。
3. 選出五個最影響使用者決策的數字，而非單純最大的數字。
4. 比對 executive summary 與視覺資料，尋找被淡化、改寫或省略的趨勢。
5. 提出一個作者最難回答但可由文件證據支持的問題。

## 證據規則

- 每個數字附頁碼、圖表名稱及原始單位。
- 區分「文件明示」、「由圖表計算」與「分析推論」。
- 圖表解析不清時明說限制，不以鄰近正文代替圖表內容。
- 計算衍生值時展示公式、分子、分母與四捨五入方式。
- 不把相關性寫成因果性。

## 輸出格式

### 文件範圍與限制

說明已涵蓋頁數、無法讀取部分及關鍵定義。

### 最重要的五個數字

使用表格：`數字 | 為何重要 | 來源頁/圖表 | 證據類型`。

### 被文字摘要淡化的趨勢

並列摘要說法、圖表所示、差距與決策影響。

### 應追問作者的問題

給出問題、提問理由及支持它的文件證據。

## 完成前檢查

- 已實際涵蓋所有可讀圖表與表格，而非只讀正文。
- 五個數字皆可追溯。
- 所有推論與文件原文清楚分開。
