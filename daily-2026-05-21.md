## 技術筆記（踩坑與 learnings）

1. **Anaconda base 的 numpy 直接 `python -c` 會 `DLL load failed (_multiarray_umath)`** — 根因同 05-20 的 PIL 問題：conda `Library\bin` 的 MKL DLL 沒掛 PATH。解：PowerShell 先 `& "...\condabin\conda-hook.ps1"; conda activate base` 再跑 python，即正常。（bash 與 PowerShell 直呼 python 都中招）
2. **專案偵測 dev server 要先查證再寫 config** — 此 workspace 是 batch 科學計算 + 靜態 deck，無任何 server。寧可如實回報「無 server」並只註冊唯一靜態檔服務，也不要在 launch.json 塞 `<server-name>` placeholder。
3. **`preview_start` 用法**：server 由 `.claude/launch.json` 的 `name` 啟動；`runtimeExecutable`=指令、`runtimeArgs`=參數；`python -m http.server` 用 `--directory` 指定服務根目錄。
4. **GP correlation dimension 判讀靠定性對比**：正控制（Rössler）收斂到 ~1.8（真值≈2，受序列長度/scaling band 略低估）就足以證明方法有效；測試組「收斂 vs 隨 m 爬升」才是 attractor vs noise 的判準，不需糾結絕對值。
5. **Obsidian 寫入**：用 mcp-obsidian `append_content`（vault 相對路徑）建 `.md`；圖檔／腳本要另外用 Bash `cp` 到 vault 實體路徑（`G:\我的雲端硬碟\Second Brain\…`，含中文路徑）。
