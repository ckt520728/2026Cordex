## 技術筆記（learnings / 踩坑）

- **Windows 推送**：Git Bash 無法非互動取得 GitHub 憑證（`credential.helper=manager` → "could not read Username"）；改走 **PowerShell**，GCM 靜默生效。PowerShell 會把 git 進度包成紅字 `NativeCommandError`，但**看最後一行 `..  <branch> -> <branch>` 才是真結果**。
- **分支名各 repo 不同**：`kidney-cognition-lab` = `main`、`claude-skills` = `master`、`2026Cordex` = `master`。push 前先 `git branch --show-current` 確認（今天就因這個吃了一次 "src refspec main does not match any"）。
- **公開頁面發佈前務必一手查證引用**（Merker 2013→2016 的更正）。
- **Drive `create_file` 只能新建不能覆寫**：改版用版本化檔名，事後手動清舊版。
- **claude-skills 慣例**：每個 skill 同時有 `<name>/SKILL.md`（原始）與 `<name>.skill`（zip 打包）兩形態；README 每項附「Portable artifact」。
