## 技術筆記

- **plugin 只能從 marketplace 裝，不能從 zip 裝。** marketplace = 一個含 `.claude-plugin/marketplace.json` 的 git repo。repo 裡的 `research-to-publication-0.1.1.zip` 只是打包產物，內容和 `plugins/research-to-publication/` 資料夾一模一樣，所以直接指向 repo 就好，zip 是多餘的（而且手動解壓縮會失去版本追蹤與 `plugin update`）。
- **版本以 `plugin.json` 為準**：安裝時解析出來的是 0.1.1，即使 marketplace entry 寫 0.1.0 也不會壞。但 `claude plugin tag` 會驗證兩者一致，不一致會擋發版，所以還是得修。
- **Windows 長路徑繞法**：`git clone --no-checkout --depth 1 --filter=blob:none` 再 `git sparse-checkout set <只要的資料夾>`，就能只取出要動的檔案，避開超長檔名。
- **`raw.githubusercontent.com` 有 CDN 快取**：merge 完馬上去讀，很可能還是舊值（今天就讀到 0.1.0）。要驗證真相請用 `gh api repos/{owner}/{repo}/contents/{path}?ref=master`，那個沒有快取。
