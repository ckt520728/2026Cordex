# Publishing contract

Target repository: `https://github.com/ckt520728/kidney-cognition-lab`, branch `main`.

Required files: `blog/<slug>.html`, `index.html`, and `sitemap.xml`.

House style must be extracted from the current repository. Use a standalone article with inline visuals. Check desktop and mobile rendering, JavaScript syntax, citations, author metadata, homepage registration, sitemap registration, repository cleanliness, push result, and public content on both deployment endpoints.

Google Drive is a snapshot, not the source of truth. A failed optional mirror does not invalidate a verified website deployment, but must be reported.
