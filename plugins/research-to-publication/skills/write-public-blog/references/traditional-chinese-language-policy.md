# Traditional-Chinese language policy

## Objective

The article should sound like professional Traditional-Chinese medical or scientific writing, not an English manuscript with Chinese connective words. Precision comes from controlled terminology, not frequent code-switching.

## First-use rule

Use one of these forms at the first substantive occurrence:

- `校準（calibration）`
- `反機率加權（inverse probability weighting, IPW）`
- `治療預測生物標記（treatment-predictive biomarker）`

Afterward, use `校準`, `反機率加權` or `IPW`, and `治療預測生物標記`. Do not repeat the full English phrase.

If an established acronym is the normal clinical form, define it once and continue with the acronym:

- `尿白蛋白／肌酸酐比值（urinary albumin-to-creatinine ratio, UACR）` → `UACR`
- `重大心血管不良事件（major adverse cardiovascular events, MACE）` → `MACE`

## English allowlist

English may remain without forced translation for:

- proper model names: `RECODe`, `SCORE2-Diabetes`, `UKPDS-OM2`, `CHIME`;
- trial names: `LEADER`, `SUSTAIN-6`, `EMPA-REG OUTCOME`, `DAPA-HF`;
- generic drug names: `liraglutide`, `semaglutide`, `empagliflozin`, `dapagliflozin`;
- genes, proteins, biomarkers, units, formulas, statistical symbols, and established abbreviations;
- titles and metadata inside the reference list;
- verbatim quotations whose wording matters.

English is not automatically allowed merely because it is common in a manuscript. Translate general research nouns and connective prose.

## Terms that normally become Chinese

After first definition, prefer these Chinese forms:

| Avoid repeated English | Preferred later form |
|---|---|
| population | 族群 |
| outcome | 結局 |
| time horizon | 預測期間 |
| discrimination | 區辨力 |
| calibration | 校準 |
| external validation | 外部驗證 |
| evidence | 證據 |
| mechanism | 機轉 |
| prognostic marker | 預後標記 |
| treatment-predictive biomarker | 治療預測生物標記 |
| benefit | 效益 |
| safety signal | 安全訊號 |
| framework | 架構 |
| trigger | 提醒訊號 |

## Audit method

Before Gate 3:

1. Read the article aloud or simulate an oral read-through; flag clauses that require switching language more than once.
2. Inspect narrative sentences containing three or more English lexical items outside the allowlist.
3. Check the first-use term table against the actual order of the article.
4. Confirm later occurrences use Chinese or the selected acronym.
5. Check headings, figure labels, captions, tables, tags, metadata, and homepage copy separately.
6. Re-run number and citation checks after the language pass.

Do not use a universal English-token threshold as a hard failure: topics differ. Use counts as before/after diagnostics, then apply the allowlist and sentence-level review.
